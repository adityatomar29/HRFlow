# HRFlow: A Simple Employee Management System 👨‍💼

A robust Employee Management System built with Django that streamlines employee record management through an intuitive and user-friendly web interface. The application enables organizations to efficiently maintain employee information, perform CRUD operations, and manage records from a centralized dashboard.

## 🚀 Live Demo

🔗 **Deployed Application:**
[Visit HRFlow](https://aditya29.pythonanywhere.com)

---

## 📖 Project Overview

The Employee Management System is designed to simplify employee data management by providing essential functionalities such as adding, updating, viewing, searching, and deleting employee records. The application follows Django's MVC architecture and includes an administrative panel for advanced management.

---

## ✨ Features

* ➕ Add new employee records
* 📝 Update employee information
* 🗑️ Delete employee records
* 👀 View all employees in a structured format
* 🔍 Search employees by relevant criteria
* 🔐 Secure Django Admin Panel
* 📱 Clean and responsive user interface
* 💾 SQLite database integration

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/adityatomar29/HRFlow.git
cd office_emp_proj
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Apply Database Migrations

```bash
python manage.py migrate
```

### 6. Create Admin User (Optional)

```bash
python manage.py createsuperuser
```

### 7. Run Development Server

```bash
python manage.py runserver
```

The application will be available at:

```text
http://127.0.0.1:8000/
```
